# AI_Agent_HorrorStory
Task : Story Generator Agent 
Build an AI agent using the ChatGPT API that generates very short horror stories, with strict 
constraints: 
System Requirements 
1. The story must NOT exceed 300 characters (not 300 words). 
2. Every time the agent is called, it must generate a unique horror story (never 
repeating previous ones). 
3. If the output exceeds 300 characters, the agent must self-correct and regenerate. 
4. The system must store past outputs and compare new generations to avoid repetition. 
5. Offers an API endpoint to request the story.
